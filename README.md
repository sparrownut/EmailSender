# EmailSender  
一个批量邮件发送器 半自用 有建议Issues中提出 很可能下个版本就会修正/添加功能  
body文件中可以使用的环境变量:  
%RECV_EMAIL% 对应接收者邮箱  
%SEND_EMAIL% 对应发送者邮箱
